## API Summary

<div class="doc-api-card card">
  <div class="doc-api-heading">
    <span class="doc-api-kind">class</span>
    <code>Engine</code>
  </div>
  <p>The root interaction controller. It owns the frame loop, input routing, and object lifecycle for a DOM region.</p>
  <div class="doc-meta-grid">
    <span>Package</span>
    <code>@snap-engine/core</code>
    <span>Import</span>
    <code>import {"{ Engine }"} from "@snap-engine/core"</code>
    <span>Lifecycle</span>
    <code>engine.element = container</code>
  </div>
</div>

## Code + Action

```ts
import { Engine } from "@snap-engine/core";

const engine = new Engine();
engine.element = document.querySelector("[data-scene]");
```
