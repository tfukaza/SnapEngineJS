SnapEngine's camera module allows implementation of zoom and pan.
It also introduces another important concept, the _camera coordinate_.

Camera coordinates start at the top left of the engine's DOM element.
It is not affected by any pan or zoom, making it useful to know the 
location of objects on the screen. 

For coordinate conversion methods, camera setup, and CameraControl examples, see [Camera reference](/docs/snapengine/reference/camera).
