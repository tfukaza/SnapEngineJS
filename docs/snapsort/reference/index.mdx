The framework selector in the sidebar keeps the current component selected when you switch between Svelte and React.

## Packages

| Package | Purpose |
|---------|---------|
| `@snap-engine/snapsort` | Framework-agnostic SnapEngine objects and types used by framework bindings. |
| `@snap-engine/snapsort-svelte` | Svelte components that wrap the core objects. |
| `@snap-engine/snapsort-react` | React components that wrap the core objects. |
