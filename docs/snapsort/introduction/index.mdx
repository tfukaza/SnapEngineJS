Install the component library for your framework.

For Svelte:

```bash
npm install @snap-engine/core @snap-engine/asset-base-svelte @snap-engine/snapsort @snap-engine/snapsort-svelte
```

For React:

```bash
npm install @snap-engine/core @snap-engine/snapsort @snap-engine/snapsort-react react react-dom
```
