# @snap-engine/snapline

Core TypeScript logic for SnapEngine node graph interfaces.

## Install

```bash
npm install @snap-engine/snapline @snap-engine/core
```

## Includes

- `NodeComponent`
- `ConnectorComponent`
- `LineComponent`
- `RectSelectComponent`

## Usage

```ts
import { NodeComponent } from "@snap-engine/snapline";
```
