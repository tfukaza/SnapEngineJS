# @snap-engine/snapline-svelte

Svelte components for SnapEngine node graph interfaces.

## Install

```bash
npm install @snap-engine/snapline-svelte @snap-engine/snapline
```

## Includes

- `Node`
- `Connector`
- `Line`
- `Select`

## Usage

```svelte
<script>
  import { Node, Connector, Line, Select } from "@snap-engine/snapline-svelte";
</script>
```
